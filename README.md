# C++ Calculator

This is a basic calculator program written in C++. 
It performs:
Addition,
Subtraction,
Multiplication,
Division using a menu-driven interface.

## How to Run

1. Compile the code:
   g++ calculator.cpp -o calculator

2. Run the program:
   ./calculator

- Using conditionals and loops effectively
